# Keyboard Shortcuts

MarketMind supports the following keyboard shortcuts for improved accessibility and productivity.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + ,` | Open Settings |
| `Cmd/Ctrl + K` | Focus Chat Input |
| `Cmd/Ctrl + B` | Toggle Chat Sidebar |
| `Cmd/Ctrl + /` | Show Keyboard Shortcuts |
| `Esc` | Close Dialogs |

## Chart Controls

| Shortcut | Action |
|----------|--------|
| `V` | Toggle Volume Display |
| `G` | Toggle Grid Display |
| `C` | Switch Chart Type (Kline/Line) |
| `1-5` | Toggle Moving Averages (MA9, MA20, MA50, MA100, MA200) |
| `+` / `=` | Zoom In |
| `-` | Zoom Out |
| `0` | Reset Zoom |
| `←` / `→` | Pan Left/Right |
| `Space` + Drag | Pan Chart |

## Chat Interface

| Shortcut | Action |
|----------|--------|
| `Enter` | Send Message |
| `Shift + Enter` | New Line |
| `↑` | Edit Last Message (when input is empty) |
| `Cmd/Ctrl + L` | Clear Chat History |

## Symbol Selector

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + P` | Open Symbol Selector |
| `↑` / `↓` | Navigate Symbol List |
| `Enter` | Select Symbol |
| `Esc` | Close Selector |

## Accessibility Features

### Screen Reader Support
- All interactive elements have proper ARIA labels
- Form inputs include descriptive labels
- Chart data is announced to screen readers
- Error messages are announced immediately

### Keyboard Navigation
- Full keyboard navigation support
- Visible focus indicators
- Skip to main content link
- Tab order follows logical flow

### Visual Accessibility
- High contrast mode support
- Respects system color preferences
- Adjustable text sizes
- Color-blind friendly indicators

### Motor Accessibility
- Large click targets (minimum 44x44px)
- Sticky keyboard modifiers supported
- No time-based interactions required
- All drag operations have keyboard alternatives

## Customization

Keyboard shortcuts can be customized in Settings → General → Keyboard Shortcuts.

## Platform-Specific Notes

- On macOS, `Cmd` is used as the primary modifier
- On Windows/Linux, `Ctrl` is used as the primary modifier
- Function keys (F1-F12) are reserved for future features

## Tips

- Hold `Shift` while using arrow keys to select text
- Use `Tab` to navigate between form fields
- Press `?` anywhere to show this help dialog
- Most shortcuts work globally within the app
